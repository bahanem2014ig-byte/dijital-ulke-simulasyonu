# Dijital Ülke Simülasyonu

İnternet verilerine dayalı yapay zeka kişiliklerinden oluşan Dijital Ülke Simülasyonu.

Her vatandaş, insanların internete yüklediği içeriklere göre karakterize edilmiş bir yapay zeka modelidir.

## Teknolojiler
- React + Vite
- TypeScript
- Express.js
- OpenAI GPT
- Tailwind CSS
- Drizzle ORM + PostgreSQL
